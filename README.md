# HZZDZZ Couple Finance v0.8 Beta

## 목표
엑셀처럼 한눈에 보는 부부 예산·정산 관리 웹앱입니다.

## 덮어쓰기 파일
- index.html
- style.css
- script.js
- firebase.js

## 주요 반영 내용
- 홈 화면을 엑셀형 표 중심 대시보드로 재설계
- 총 자산 현황을 홈 최상단에 배치
- 월별 수입/고정지출/예산사용/잉여자금 표기
- 예산현황 표: 식비(월예산), 생필품/비상금/쇼핑비/부모님/경조사비/육아(연예산)
- 자투리 통장은 예산현황 영역으로 이동
- 공동지출 내역 화면 추가 및 수정/삭제 가능
- 정산 자동 계산
- 월별 고정지출 입력 구조
- 다혜 급여: 1월~12월 당직비 입력, 세후 예상 월급 계산
- 투자관리: 국내주식/해외주식/CMA
- Firebase 공동 동기화 유지

## Firebase
Firestore Database가 생성되어 있어야 합니다.
두 사용자 모두 같은 Firebase 설정값과 같은 가계부 ID(hzzdzz_가계부)를 입력해야 공동 동기화됩니다.
