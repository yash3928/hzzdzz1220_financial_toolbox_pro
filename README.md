HZZDZZ Couple Finance v1.2.3

변경 사항
- 연도/월 선택은 각 휴대폰의 localStorage에만 저장됩니다.
- 예산, 지출, 급여, 고정지출, 자산, 투자 데이터만 Firebase로 공동 동기화됩니다.
- 한 사용자가 연도/월을 바꿔도 상대방 화면은 바뀌지 않습니다.
- 중복된 js/, css/ 폴더와 wrapper 파일을 제거했습니다.
- 루트 4개 파일만 사용합니다: index.html, app.js, firebase.js, style.css

업로드
- 압축을 풀고 위 4개 파일을 GitHub 저장소 루트에 덮어쓰세요.
- 기존 js/ 폴더, css/ 폴더, script.js는 삭제해도 됩니다.
