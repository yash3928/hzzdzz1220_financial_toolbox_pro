# HZZDZZ Couple Finance v0.9.0

## 핵심 변경
- 홈 화면 UI 기준 통일: 아이콘, 표, 버튼, 접기/펼치기 형식 정리
- 총 자산 현황은 홈 최상단에 항상 표시
- 월별 지출 / 월급 현황 / 예산 현황 / 투자 현황 / 수입·고정지출은 컴팩트 접기/펼치기 방식 적용
- 자산 구조 변경
  - 현금은 사용자가 세부 분류를 추가/수정/삭제
  - 국내주식/해외주식/CMA는 투자 화면 입력값이 자산에 자동 반영
  - 여행비/연금/청약/코인/기타는 자산에서 직접 입력
- 투자 구조 단순화
  - 국내주식: 현재금액 + 수익률
  - 해외주식: 현재금액 + 수익률
  - CMA: 현재잔액
  - 매수/매도/종목별 거래내역은 관리하지 않음
- 기존 Firebase 데이터 보존 구조 유지
