## 주요 변경
- 홈 화면을 재무 대시보드 중심으로 개편
- 공동예산 분류: 식비, 생필품, 의료, 비상금, 여행비, 경조사비, 육아
- 공동예산은 클릭 시 세부 분류 표시
- 월별 공동예산 설정 기능 추가
- 매월 고정지출 관리 추가
- 연간 계획 예산 관리 추가
- 날짜 입력칸 모바일 잘림 수정
- 자산현황에서 ETF 삭제, 국내주식/해외주식은 투자관리와 자동 연동
- 투자 종목별 원금/평가금액/개별 수익률/총 수익률 표시
- 진혁 월급 직접 입력, 다혜 연간 당직표 기반 월급 계산
- 월 기준일 기본 10일, 사용자가 변경 가능
- Firebase 공동 동기화 유지

## 사용 순서
1. 설정에서 Firebase 공동 동기화가 연결되어 있는지 확인합니다.
2. 예산 메뉴에서 월별 공동예산을 먼저 입력합니다.
3. 매월 고정지출과 연간 계획 예산을 입력합니다.
4. 입력 메뉴에서 공동지출과 월급을 입력합니다.
5. 자산 메뉴에서 자산과 투자 종목을 입력합니다.
6. 홈 화면에서 예산, 자산, 투자, AI 요약을 확인합니다.
