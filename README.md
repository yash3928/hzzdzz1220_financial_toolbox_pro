# HZZDZZ Couple Finance v0.6 Beta

## 이번 버전 핵심
- 프로그램 파일과 사용자 데이터를 분리하는 구조로 변경했습니다.
- 입력 데이터는 Firebase `households/{가계부코드}/app/data`에 저장됩니다.
- 프로그램을 업데이트해도 Firebase 데이터는 유지됩니다.
- 로컬 저장 키를 버전별 키가 아닌 고정 키로 변경했습니다.
- 기존 v0.5/v0.4 로컬 데이터는 자동으로 읽어옵니다.
- 설정에 JSON 백업/복원 기능을 추가했습니다.

## 덮어쓰기 파일
아래 4개 파일을 기존 GitHub 저장소에 업로드해서 덮어쓰기 하세요.

- `index.html`
- `script.js`
- `style.css`
- `firebase.js`

## 업데이트 전 권장
1. 현재 앱에서 설정 → 데이터 백업 / 복원 → `데이터 백업 JSON 저장`을 누릅니다.
2. GitHub에 v0.6 파일을 덮어씁니다.
3. 앱을 새로고침합니다.
4. 공동 동기화가 연결되면 기존 Firebase 데이터가 다시 표시되는지 확인합니다.

## 주의
- `현재 기기 로컬 데이터 초기화`는 Firebase 데이터를 삭제하지 않습니다.
- Firebase 데이터를 완전히 바꾸고 싶을 때만 JSON 복원을 사용하세요.
